# KiCon 2023 Asia Badge Firmware

## Build
1. Download and Install [ARM GNU Toolchain](https://developer.arm.com/Tools%20and%20Software/GNU%20Toolchain)
2. Setup Toolchain environment path
3. (optional) Install pyOCD
4. make
